 Bconomy Privacy Policy

We try to minimize the amount of data we collect and store. This is here just to let you know what we need to operate.

We are fully compliant with all EU General Data Protection Regulations (GDPR)

Website (bcono.my)

No visitor information is tracked. Period.

Bconomy Discord Bot

Bconomy collects the mininal information needed for it to function. No data is collected otherwise. When registering, this information is limited to: Your Discord User ID, Discord Username, and current Discord Server ID. No other identifiable information is collected from you. Your information is never shared to any third-parties unless explicitly stated. Any information manually supplied to the bot through commands or input (for example, names for factions or pets, channel IDs when the channel whitelist command is used) is stored as it is required for bot function. Account deletion can be requested at Discord server. Account removal will permanently delete all personal information linked to an account within 30 days of a request.

Message Filter Discord Bot

Message Filter collects the information needed for it to function when it is set up in a server. No data is collected otherwise. This information is limited to: Your Discord Server ID and Discord Channel ID(s). No message content is ever stored by the application. Your information is never shared to any third-parties unless explicitly stated. Any information manually supplied through input (for example, word filters) are stored for usage. If you wish to delete all data simply delete all filters in the application. Removing the bot from your Discord server will cause for automatic data deletion within 30 days. Contact us in our Discord server.
